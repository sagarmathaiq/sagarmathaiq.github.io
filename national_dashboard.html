Methodology

This forecast uses a core+contestable framework where each party has an irreducible organizational base (core) plus a contestable portion driven by national latent factors. RSP's core is set at 20% nationally and uplifted by 1.75× in districts to reflect organizational expansion from 2079 to 2082. The model projects both PR (110 seats) and FPTP (165 seats) outcomes using the same underlying national mood factors, ensuring coherent chamber forecasts.
